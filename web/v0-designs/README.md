# v0 Designs

This folder contains UI components exported from [Vercel v0](https://v0.dev).

## How to Use

1. Export components from v0 and paste them here
2. Name files descriptively: `PantryList.tsx`, `Dashboard.tsx`, `AddItemModal.tsx`
3. The frontend agent will integrate these with real API data

## Design Guidelines

See `/docs/UI_PROMPTS_V0.md` for the prompts used to generate these designs.

### Style Reference
- Pastel pink: `#FFB5C5`
- Pastel mint: `#B5EAD7`
- Pastel lavender: `#C9B5E8`
- Pastel peach: `#FFDAB3`
- Cream background: `#FFF9F5`
